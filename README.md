# Typography Art Generator

入力したテキストをランダムなデザインとエフェクトで再構築し、アートとして生成・保存できるWebアプリケーションです。

## 主な機能

*   **動的タイポグラフィ生成**: 入力されたテキストを単語単位で分解し、フォントやスタイルをランダムに適用します。
*   **カオス度調整スライダー**: リアルタイムでアートの「カオス度（サイズ、回転、色相、エフェクト）」を調整可能です。
*   **多彩なエフェクト**: 影（Shadow）、ネオン（Glow）、ぼかし（Blur）、中抜き（Hollow）といった効果がランダムに割り当てられます。
*   **永続保存**: 生成したアートはデータベース（SQLite）に保存され、後からギャラリー形式で閲覧可能です。
*   **レスポンシブデザイン**: Tailwind CSSによるモバイル対応の洗練されたUIです。

## 技術スタック

*   **Backend**: Python, Flask
*   **Frontend**: HTML5, Tailwind CSS, JavaScript (Vanilla)
*   **Database**: SQLite
*   **Text Processing**: Janome (日本語形態素解析)

## 必要な環境

Pythonがインストールされている環境で、以下のライブラリをインストールしてください。

```bash
pip install flask janome
```
